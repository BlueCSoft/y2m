收款员号
收款员密码
二维码
终端号
  
地址:http://101.200.196.21/gmp/bin/mobileiface/login.jsp

参数:
counterid 终端号(专柜号),
cashierid 收银员号,
pwd 密码,
barcode 二维码

收银员号+密码和二维码至少输入一个。

{

  "storeid":"0001",
 "counterid":"010A1",

 "code":"1",
 "msg":"sucess"
 "sub_code":"1",

}      

返回说明:
  storeid   : 门店号,
  counterid : 终端号,
  cashierid : 收银员号,
  code      : 网络协议代码 1成功，0失败
  msg       : 网络异常信息
  sub_code  : 业务代码 1成功，0失败
  sub_msg   : 业务处理异常信息,
  cashier   : 成功时为收银员名称
  time      : 服务器时间
